# Manga Reader App — Project Hub

<aside>

**Goal (MVP):** Arabic-first (RTL) cross-platform manga/manhwa/manhua reader (Flutter), backendless, using MangaDex + Jikan. Local storage for favorites/progress/cache.

</aside>

## 1) Overview

- **Tech:** Flutter (Clean Architecture + feature-first modular)
- **APIs:** MangaDex, Jikan
- **Local DB:** Drift or Isar
- **Languages:** Arabic (default, RTL) + English (fallback)
